# Kickpad

Generate, tweak and evolve kick drum sounds and audio samples.

## Requirements

### Run-time

* `ffplay`

### Build-time

* `go` (1.23 or later)

## Screenshot

## Installation

    go install github.com/xyproto/kickpad@latest

## Running

    kickpad
